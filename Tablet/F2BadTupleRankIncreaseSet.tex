\begin{definition}
For a \(2k\)-tuple
\[
  ab=(a_1,b_1,\ldots,a_k,b_k)\in(\mathbb F_2^p)^{2k},
\]
let \(r_i\) denote the rank quantity from \(\noderef{F2BadTupleRank}\). The
rank-increase set of \(ab\) is the set of zero-based indices
\[
  I(ab)=\{\,i\in\{0,\ldots,k-1\}: r_{i+1}=r_i+1\,\}.
\]
\end{definition}
