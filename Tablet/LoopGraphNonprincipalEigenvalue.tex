\begin{definition}
For a finite loop graph \(G\), a real number \(\mu\) is a non-principal adjacency eigenvalue if there is a nonzero real-valued function \(f\colon V(G)\to \mathbb R\) whose coordinate sum is zero and which satisfies
\[
  A_G f = \mu f.
\]
For a regular symmetric loop graph, these are the eigenvalues other than the all-ones eigenvalue.
\end{definition}
