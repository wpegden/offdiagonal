\begin{definition}
For a vertex type \(V\), a directed graph, or digraph, is a binary arc relation
\[
  A\colon V\to V\to \mathrm{Prop}.
\]
The assertion \(A(x,y)\) means that there is an arc from \(x\) to \(y\).
This definition allows loops; looplessness is imposed by a separate predicate when the paper requires it.
\end{definition}
